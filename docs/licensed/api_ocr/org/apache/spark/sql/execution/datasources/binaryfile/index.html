  1.  class BinaryFileFormatReadWrite extends BinaryFileFormat

    The binary file data source.

    The binary file data source. It reads binary files and converts each file into a single record that contains the raw content. and metadata of the file. It writes image/pdf/text object into a local/hdfs/s3 path Example:

    // Scala
  val df = spark.read.format("binaryFormat")
     .load("/path/to/fileDir")


     df.write.format("binaryFormat")
      .option("type", "image")
      .option("field", "image")
      .option("prefix", "ocr_")
      .option("extension", "jpeg")
      .mode(SaveMode.Overwrite)
      .save("s3a://path/to/fileDir")
