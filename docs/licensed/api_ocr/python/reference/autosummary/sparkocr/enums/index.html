class AdaptiveThresholdingMethod#

Bases: enum.Enum

Method used to determine adaptive threshold for local neighbourhood in

weighted mean image.

  • ‘gaussian’: apply gaussian filter

  • ‘mean’: apply arithmetic mean filter

  • ‘median’: apply median rank filter

MEAN = 'mean'#
MEDIAN = 'median'#
SINGH = 'singh'#
WOLF = 'wolf'#
