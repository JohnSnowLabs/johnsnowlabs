redact_phi_in_tiles(svs_path, tiles_deid, main_folder, output_svs_path=None, create_new_svs_file=False)#

Modifies tiles in an SVS file to remove sensitive information by drawing black rectangles. Can operate on a single SVS file or a directory containing multiple SVS files. The function will first scale the detections to all pyramid levels before applying deidentification. Parameters: - svs_path (str): Path to a single SVS file or a directory containing SVS files. - tiles_deid (list or DataFrame): List of dictionaries or a PySpark DataFrame containing the de-identification instructions. - main_folder (str): Path to folder containing tile directories. - output_svs_path (str, optional): Path where the new de-identified SVS file(s) will be saved. - create_new_svs_file (bool): If True, creates new SVS files instead of modifying originals. - max_group_mem_gb (float): Max memory per group for batching files.
