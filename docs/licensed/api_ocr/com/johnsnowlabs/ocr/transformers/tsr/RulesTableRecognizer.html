class RulesTableRecognizer extends TableRecognizer

Table recognizer that based on rules and heuristics. It is based on a number of assumptions - Provided data contains only one table data - Provided data contains full table data (without cut) - Provided data does not contain data that are out of table - There is only one header row and it is the upper row - The first column is the index column Processing flow is decomposed on number of steps: - Initial rows detection - Order text regions in row by x coordinates left-to-right - Initial text regions distribution on columns - Split incorrectly joined columns - Join rows that misses index with one that have
