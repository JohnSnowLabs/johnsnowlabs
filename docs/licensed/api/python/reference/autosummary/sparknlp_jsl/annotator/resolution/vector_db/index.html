class VectorDBApproach#

Bases: sparknlp_jsl.common.AnnotatorApproachInternal, VectorDBApproachParams

Contains all the methods for training a VectorDBModel model. This annotator returns the top similar documents which are stored in a VectorDB model via the VectorDBApproach. After the calculation of the top similar documents, It returns the top similar documents as VECTOR_SIMILARITY_RANKINGS output annotator type. Additionally, It returns distance, score, target_text, code, resolved_text, and saved other metadata fields in the output metadata. Ideally this annotator works in conjunction with DocumentAssembler, DocumentSplitter, and SentenceEmbeddings.

Input Annotation types

Output Annotation type

DOCUMENT, SENTENCE_EMBEDDINGS

VECTOR_SIMILARITY_RANKINGS

class VectorDBModel(classname='com.johnsnowlabs.nlp.annotators.resolution.VectorDBModel', java_model=None)#

Bases: sparknlp_jsl.common.AnnotatorModelInternal, VectorDBModelParams, sparknlp_jsl.common.HasStorageRef

Annotator that returns the top similar documents which are stored in a VectorDB model via the VectorDBApproach. Input annotator types: SENTENCE_EMBEDDINGS and DOCUMENT. DOCUMENT is a query identifier. And SENTENCE_EMBEDDINGS is the embeddings of the query. After the calculation of the top similar documents, It returns the top similar documents as VECTOR_SIMILARITY_RANKINGS output annotator type. Additionally, It returns distance, score, target_text, code, resolved_text, and saved other metadata fields in the output metadata.

Input Annotation types

Output Annotation type

DOCUMENT, SENTENCE_EMBEDDINGS

VECTOR_SIMILARITY_RANKINGS
